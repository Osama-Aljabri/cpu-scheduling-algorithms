# 🧠 CPU Scheduling Algorithms

## 📌 Overview
This project demonstrates the implementation of key CPU scheduling algorithms used in operating systems to manage process execution efficiently. It simulates how processes are scheduled and evaluates their performance.

---

## ⚙️ Algorithms Implemented
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Round Robin (RR)

---

## 🎯 Objectives
- Understand how CPU scheduling works
- Compare different scheduling algorithms
- Analyze performance using:
  - Waiting Time
  - Turnaround Time
